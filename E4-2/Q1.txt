問1：

  貸出記録テーブル(rental)上に
  貸出日をキーにしたインデックスを作成して見ましょう。
  以下の空欄を埋めて,SQL命令を完成させてください。

  (①)
    ind_rental
  (②)
    rental
    (
      (③)
    )
  ;


回答：

  ①：CREATE INDEX
    ind_rental
  ②：ON
    rental
    (
      ③：rental_date
    )
  ;


結果：

  CREATE INDEX       -- 貸出記録テーブルにインデックスを設定する
    ind_rental       -- インデックス名を設定する
  ON                 -- 以下テーブルのフィールドをインデックスとして設定します
    rental           -- 貸出記録テーブルの
    (
      rental_date    -- 貸出日フィールドをインデックスとして設定します
    )
  ;

  +--------+------------+------------+--------------+-------------+-----------+-------------+----------+--------+------+------------+---------+---------------+

  +--------+------------+------------+--------------+-------------+-----------+-------------+----------+--------+------+------------+---------+---------------+
  | rental |          0 | PRIMARY    |            1 | id          | A         |           0 |     NULL | NULL   |      | BTREE      |         |               |
  | rental |          1 | ind_rental |            1 | rental_date | A         |           0 |     NULL | NULL   | YES  | BTREE      |         |               |
