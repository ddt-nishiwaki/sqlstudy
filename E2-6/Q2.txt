問２：

  ユーザテーブル(usr)とアンケート回答テーブル(quest)、
  それぞれに登録されているユーザ氏名を比較し、
  ユーザテーブル(usr)にしか登録されていないもののみを取り出して見ましょう。
  以下の空欄を埋めて、SQL命令を完成させてください。

    SELECT
      (①)
    FROM
      usr AS u
  (②)
    SELECT
      q.name
    FROM
      (③)
  ;


回答：
  ※mySQLがEXCEPT演算子に対応していないためサブクエリでの対応とする

  ①： CONCAT(u.l_name, u.f_name)
  ②： WHERE
        CONCAT(u.l_name, u.f_name) NOT IN
        (
  ③：   quest AS q
      )


結果

  SELECT                                -- ユーザテーブルに登録されている人の名前を取り出すために以下カラムを設定する
    CONCAT(u.l_name, u.f_name)          -- 氏名カラムを設定する
  FROM                                  -- ユーザ情報を取得するために以下テーブルから抽出する
    usr AS u                            -- ユーザテーブルから抽出する
  WHERE                                 -- アンケート回答テーブルに存在しない人の名前を取り出すために以下条件で絞り込む
    CONCAT(u.l_name, u.f_name) NOT IN   -- ユーザテーブルの氏名カラムに存在しない名前で絞り込むために以下クエリーを指定する(EXCEPT非対応のため)
    (
    SELECT                              -- アンケート回答テーブルから氏名情報を取り出すために以下カラムを設定する
      q.name                            -- 氏名カラムを設定する
    FROM                                -- アンケート回答テーブルの情報を取り出すために以下テーブルより抽出する
      quest AS q                        -- アンケート回答テーブルを指定する
    )
  ;
